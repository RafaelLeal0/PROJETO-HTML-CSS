O que é HTML?

Introdução ao HTML

HTML é uma linguagem de marcação utilizada para criar a estrutura de páginas da web. Ela organiza e define o conteúdo da página através de elementos como cabeçalhos, parágrafos, links, imagens, listas, e formulários. HTML é fundamental para a construção de qualquer site, fornecendo a base sobre a qual outros componentes, como CSS e JavaScript, são adicionados.

Tags HTML Básicas

Parágrafos

No HTML, parágrafos são definidos usando o elemento < p >. Eles são utilizados para agrupar blocos de texto em unidades lógicas separadas, facilitando a leitura e a organização do conteúdo. Cada parágrafo é criado colocando o texto entre as tags de abertura < p > e de fechamento < /p >. Parágrafos são elementos de bloco, o que significa que eles ocupam toda a largura disponível e são exibidos em uma nova linha na página.

Listas

  • Uma lista não ordenada em HTML é uma lista de itens onde a ordem dos itens não é relevante. Ela é definida usando o elemento < ul >, que significa "unordered list". Cada item dentro da lista é definido com o elemento < li >, que significa "list item". As listas não ordenadas são geralmente renderizadas com marcadores (bullets) antes de cada item.
  1. Uma lista ordenada em HTML é uma lista de itens onde a ordem dos itens é importante. Ela é definida usando o elemento < ol >, que significa "ordered list". Cada item dentro da lista é definido com o elemento < li >, que significa "list item". As listas ordenadas são renderizadas com números ou letras antes de cada item, indicando sua posição na lista.
