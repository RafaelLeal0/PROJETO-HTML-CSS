O que é CSS?

Introdução ao CSS

CSS é uma linguagem de estilo utilizada para definir a aparência e o layout de páginas da web. Ela separa o conteúdo da apresentação, permitindo que os desenvolvedores apliquem estilos como cores, fontes, espaçamentos, tamanhos e layouts ao conteúdo estruturado pelo HTML. O CSS torna as páginas da web visualmente atraentes e facilita a manutenção e a consistência do design em todo o site.

Conceitos Básicos

Seletores

Seletores são usados para "selecionar" os elementos HTML que você deseja estilizar.

  • O seletor de tag, também conhecido como seletor de elemento, é usado para selecionar todos os elementos de um tipo específico na página. Ele é definido pelo nome da tag HTML diretamente no CSS. Por exemplo, se você deseja estilizar todos os elementos < p > (parágrafos) na sua página, você usaria o seletor de tag p.

  • O seletor de classe é usado para selecionar todos os elementos que possuem uma classe específica. As classes são definidas com um ponto . seguido pelo nome da classe no CSS. Esse seletor permite aplicar estilos a um grupo de elementos, independentemente do seu tipo. Por exemplo, se você deseja estilizar todos os elementos com a classe destacado, você usaria . destacado no CSS.

  • O seletor de ID é usado para selecionar um elemento específico que possui um identificador único. Os IDs são definidos com uma cerquilha # seguida pelo nome do ID no CSS. Como um ID deve ser único em uma página, o seletor de ID é usado para aplicar estilos a um único elemento específico. Por exemplo, para estilizar um elemento com o ID cabeça, você usaria #cabeça no CSS.

Propriedades e Valores

As propriedades em CSS são os aspectos específicos dos elementos HTML que podem ser estilizados. Elas definem o que será modificado em um elemento, como cor, fonte, espaçamento, layout, entre outros. Cada propriedade tem um conjunto de valores possíveis que determinam como a modificação será aplicada.

Os valores em CSS especificam a configuração desejada para a propriedade aplicada. Cada propriedade aceita um ou mais tipos de valores, como números, palavras-chave, comprimentos, cores, URLs, entre outros. A combinação de propriedades e valores permite a criação de estilos detalhados e complexos para elementos HTML.

Classes e IDs

Classes são seletores em CSS usados para aplicar estilos a um grupo de elementos. Eles são definidos com um ponto (.) seguido por um nome de classe e são usados para compartilhar estilos entre vários elementos, permitindo uma maior reutilização de código e consistência no design. As classes são particularmente úteis quando se deseja aplicar o mesmo estilo a diferentes tipos de elementos em um documento HTML.

IDs são seletores em CSS usados para identificar um único elemento na página. Eles são definidos com um símbolo de cerquilha (#) seguido por um nome de ID e devem ser únicos dentro de um documento HTML. Os IDs têm uma especificidade maior em comparação com classes, o que significa que as regras de estilo associadas a IDs prevalecem sobre as regras de classes quando há conflitos.

Cascata e Especificidade

A cascata em CSS refere-se ao mecanismo pelo qual o navegador aplica as regras de estilo a elementos HTML quando há múltiplas regras em conflito. Esse processo é baseado em três princípios fundamentais: origem da folha de estilo, especificidade e ordem de aparição
